# Module 5 - Interactive Sales Report

Project developed as part of the DIO Power BI Analyst bootcamp (Module 5).

## Objective
Build an interactive 2-page sales management report using the Financial Sample dataset, entirely in Power BI Service (browser) without access to Power BI Desktop (Mac limitation).

## Pages
- **Page 1 - Sales Report:** KPI cards, date slicer, area chart, donut/bar chart toggle, treemap/map toggle using bookmarks and buttons
- **Page 2 - Detailed Profit Report:** Waterfall chart, bar chart, line chart, treemap, year/country slicers, quarterly profit analysis

## Features
- Navigation buttons connecting both pages
- Bookmark-based visual toggles
- Fully built in Power BI Service (browser)

## Tools
- Power BI Service (browser)
- Dataset: Financial Sample (Microsoft)
